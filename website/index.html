✨ Features

🔐

Authentication System

  • ✅ Firebase Authentication with custom claims
  • ✅ Multi-Role Support: Student, Faculty, Admin
  • ✅ Glitter-Style Login Page with animations
  • ✅ Role-based access control
👨‍🎓

Student Dashboard

  • ✅ Mood Emoji Tracker with 8 emotions
  • ✅ Attendance Button with late detection
  • ✅ Mood History Log with timestamps
  • ✅ Beautiful Framer Motion animations
👨‍🏫

Faculty Dashboard

  • ✅ Student Overview with real-time data
  • ✅ Comment System for observations
  • ✅ Alert System to notify admin
  • ✅ Search and Filter functionality
👨‍💼

Admin Panel

  • ✅ Live Face Emotion Detection
  • ✅ Real-time Camera Feed
  • ✅ Faculty Alert Management
  • ✅ MongoDB Integration & Analytics

🛠️ Quick Setup

📋 Prerequisites

  • ✅ Node.js (v16 or higher)
  • ✅ MongoDB (local or Atlas)
  • ✅ Firebase project
  • ✅ Git

⚡ Quick Start

# Clone repository
git clone <repo-url>
cd student-emotion-tracker
# Frontend setup
npm install
npm start
# Backend setup
cd ../backend
npm install
npm run dev
